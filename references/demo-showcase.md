# Demo Showcase

## Hacker News browser demo
Use this to show the platform quickly on a public, stable site.

### Setup
- `hn_demo_setup.py <task_dir>`

### What it creates
- `orchestrator_state.json`
- `runbook.json`
- `demo_guide.json`

### Why this is useful
It gives a clean, repeatable public demo for:
- pagination
- runbook mode
- orchestrator mode
- detail-batch follow-up potential

## Clean package build
Use:
- `build_clean_package.py <skill_dir> <output_dir>`

This excludes demo logs and cache junk so the distributable skill feels cleaner and more professional.
