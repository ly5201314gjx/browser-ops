# Device Profiles

Use device profiles to make browser workflows more realistic and consistent across tasks.

## Profiles
- `desktop-default`
- `mobile-default`
- `tablet-default`

## Scripts
- `device_profile_manager.py <task_dir> <profile_name>`
- `browser_human_mode.py enable <task_dir> [mode] [realism]`
- `browser_human_mode.py plan <task_dir>`

## Purpose
- standardize viewport assumptions
- standardize interaction pacing
- improve human-assisted browser handoffs
- support more realistic stepwise execution without bypassing protections
